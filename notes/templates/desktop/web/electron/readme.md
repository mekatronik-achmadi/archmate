# Run NodeJr

## Setup electron from ArchLinux

Install Electron package

```sh
sudo pacman -S electron22 trash-cli
```

## Setup electron from NPM (Recommended)

Install Electron module

```sh
npm install electron
```

## Run main script

### Using Archlinux package

```sh
electron22 main.js
```

### Using NPM

```sh
npm exec electron .
```
